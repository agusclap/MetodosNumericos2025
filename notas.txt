Ej. pseudocodigo de newtonraphson
Entrada: x0, tolerancia tol, máximo de iteraciones Nmax, paso h
Definir f(x) = 2x + ln(x) - sin(3x)

Para k = 1 hasta Nmax:
    Calcular derivada aproximada:
        df = [3*f(x0) - 4*f(x0 - h) + f(x0 - 2h)] / (2*h)

    Si |df| es muy pequeño:
        Mostrar "derivada ~ 0, no se puede continuar"
        Salir

    Calcular nueva aproximación:
        x1 = x0 - f(x0) / df

    Calcular error = |x1 - x0|

    Mostrar iteración, x0, x1, error

    Si error < tol:
        Salir con éxito

    Actualizar:
        x0 = x1

Fin Para

Salida: raíz aproximada x1
